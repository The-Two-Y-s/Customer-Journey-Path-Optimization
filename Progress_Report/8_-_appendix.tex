\addcontentsline{toc}{chapter}{APPENDIX A: SOURCE CODE AVAILABILITY}

The implementation of the \textit{Customer Journey Path Optimization} framework was developed in \textbf{Python~3.13}. The complete source code is publicly available at:

\begin{center}
\textbf{\url{https://github.com/The-Two-Y-s/Customer-Journey-Path-Optimization}}
\end{center}

\noindent The repository is organised into the following components:

\begin{itemize}[nosep]
  \item \textbf{Core algorithms} (\texttt{src/dijkstra.py}) --- baseline Dijkstra with lazy deletion and the probability-pruned variant; both return structured results containing optimal path, log-cost, path probability, and five performance metrics.
  \item \textbf{Graph construction} (\texttt{src/graph\_builder.py}) --- builds a directed weighted graph from preprocessed transition data using an adjacency-list representation.
  \item \textbf{Preprocessing} (\texttt{src/preprocessing.py}) --- ingests raw clickstream CSV data and computes maximum-likelihood transition probabilities.
  \item \textbf{Critical-$\tau$ analysis} (\texttt{src/critical\_tau.py}) --- adaptive fraction-based sweep to identify the largest $\tau^*$ preserving path optimality.
  \item \textbf{Synthetic generators} (\texttt{data/graph\_generator.py}) --- Erd\H{o}s--R\'{e}nyi and layered stage-based graphs with BFS connectivity guarantees and deterministic \texttt{hashlib.md5} seeding.
  \item \textbf{Real-data loaders} (\texttt{data/real\_data\_loader.py}) --- parsers for RetailRocket (event-level and item-level) and RecSys~2015 YOOCHOOSE datasets.
  \item \textbf{Experiment runners} (\texttt{run\_experiments.py}, \texttt{run\_real\_experiments.py}) --- execute the full parameter matrix (2,160 synthetic + 840 real-data configurations).
  \item \textbf{Analysis notebook} (\texttt{analysis.ipynb}) --- generates all plots and statistical tests (Wilcoxon signed-rank) from experiment CSV files.
\end{itemize}

\noindent Installation instructions, reproduction commands, and usage details are documented in the repository README.
